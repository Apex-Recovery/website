## ⚡ Features

- **Material 3 Design:** Clean, fluid UI with light/dark mode support, fine-tuned animations, and backdrop blur effects.
- **Dynamic Downloads:** Devices and builds are fetched on the fly directly via the **GitHub API Releases**-no hardcoded device lists needed.

---

## ⚙️ Tech Stack

- **Framework:** React + Vite (SPA architecture)
- **Routing:** `wouter` (minimalist, high-performance client-side router)
- **Styling:** Tailwind CSS v4 + Shadcn UI primitive components
- **Animations:** `framer-motion`
- **Data Fetching:** `@tanstack/react-query` (handling 5-minute automated caching and stale-data states)
